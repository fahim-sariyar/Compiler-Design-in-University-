# Compiler Design Lab

## Experiment Name
Vowel and Consonant Detection using Lex

## Objective
To identify whether an input character is a vowel or consonant using lexical analysis.

## File Name
`vowel.l`

## How to Run

```bash
lex vowel.l
gcc lex.yy.c -o vowel
./vowel
```

## Sample Input
a
b
k

## Sample Output
a is a vowel
b is a consonant
k is a consonant
